# ?? Interactive Student Directory

Hey there! This is my first big React project! I built this interactive student directory to help manage a classroom list. It's got search, filtering, and you can even mark people present or absent. It looks pretty cool too!

## ?? Features

Basically, I added everything that was asked for:
- **Add Students:** You can fill out a form to add someone new.
- **Search & Filter:** Search by name or filter by which course they're in.
- **Sorting:** Sort by name or grades (to see the top performers!).
- **Status Toggles:** A button to mark someone "Present" or "Absent".
- **Delete:** If someone leaves, you can remove them.
- **Top Performer Badge:** If someone has a grade of 90 or more, they get a special star badge!
- **LocalStorage:** It saves your data even if you refresh the page! (I was so happy when I got this working).

## ??? How to run it

I used Vite for this, so it's super fast. Here is how you get it running on your computer:

1.  **Install dependencies:**
    ```bash
    npm install
    ```
2.  **Start the development server:**
    ```bash
    npm run dev
    ```
3.  **Open it in your browser:**
    Click the link that shows up in your terminal (usually `http://localhost:5173`).
